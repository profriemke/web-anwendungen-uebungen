Aufgabe C.3.2: Nachrichten-Teaser mit CSS

Budget: 20 Minuten

+------------------------------------------------------+
|                       .teaser                        |
|  +-----------+   +-----------------------------+     |
|  |           |   | .teaser-title               |     |
|  |  IMAGE    |   | Stadt testet neues ...      |     |
|  |           |   +-----------------------------+     |
|  |           |   | .teaser-text                |     |
|  +-----------+   | In der Innenstadt ...       |     |
|                  | ...                         |     |
|                  |                             |     |
|                  +-----------------------------+     |
|                                                      |
|                  +-----------------------------+     |
|                  | .content                    |     |
|                  | weiterer Seiteninhalt       |     |
|                  +-----------------------------+     |
+------------------------------------------------------+

Sie gestalten eine Nachrichten-Seite mit CSS. Auf dieser Seite wird eine Nachhricht angeteasert.

Das Layout entspricht der folgenden Skizze:

  • Ein Element .teaser.
  • .teaser enthält .teaser-title und .teaser-text
  • Links neben .teaser ist ein Teaser-Bild (.teaser-image)
  • Unter dem .teaser steht noch weiterer Content in .content.

Setzen Sie dieses Layout um. Sie können hierzu Floating, Grid oder Flexbox verwenden. Die Musterlösung verwendet Flexbox.
